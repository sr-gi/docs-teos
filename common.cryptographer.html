static decrypt(encrypted_blob, secret)[source]¶

Decrypts a given EncryptedBlob using CHACHA20POLY1305.

SHA256(secret) is used as key, and 0 (12-byte) as iv.

Parameters

  • encrypted_blob (EncryptedBlob) – an EncryptedBlob potentially containing a penalty transaction.

  • secret (str) – a value to used to derive the decryption key. Should be the dispute txid.

Returns

The decrypted data (hex encoded).

Return type

str
