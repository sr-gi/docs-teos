find_last_common_ancestor(last_known_block_hash)[source]¶

Finds the last common ancestor between the current best chain tip and the last block known by us (older block).

This is useful to recover from a chain fork happening while offline (crash/shutdown).

Parameters

last_known_block_hash (str) – the hash of the last know block.

Returns

A tuple (str, list) where the first item contains the hash of the last common ancestor and the second item contains the list of transactions from last_known_block_hash to last_common_ancestor.

Return type

tuple

is_block_in_best_chain(block_hash)[source]¶

Checks whether a given block is on the best chain or not. Blocks are identified by block_hash.

A block that is not in the best chain will either not exists (block = None) or have a confirmation count of -1 (implying that the block was forked out or the chain never grew from that one).

Parameters

block_hash (str) – the hash of the block to be checked.

Returns

True if the block is on the best chain, False otherwise.

Return type

bool

Raises

KeyError – If the block cannot be found in the blockchain.
